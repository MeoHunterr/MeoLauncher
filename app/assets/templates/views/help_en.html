1. Getting Started

MeoLauncher is designed to simplify playing Minecraft Beta 1.7.3 with modern features.

System Requirements

  • Java: Java 8 or newer (Automatically detected)
  • RAM: Minimum 1GB, Recommended 2GB-4GB
  • OS: Windows 10/11 (64-bit)

Main Interface

  • Home: Login and launch the game
  • Screenshots: Manage in-game screenshots
  • Texture Packs: Install and manage resource packs
  • Skin: Manage Ely.by skins and capes
  • Settings: General launcher and game settings
  • Performance: Advanced optimization tweaks

2. Accounts & Login

MeoLauncher supports 3 login modes:

Microsoft (Premium)

  • Click Login with Microsoft on the Home screen
  • Follow instructions to authenticate via browser
  • Supports official skins and premium servers

Ely.by (Skin System)

  • Click Login with Ely.by on the Home screen or go to the Skin tab
  • Enter your Ely.by username/email and password
  • Supports skins, capes, and friends list from the Ely.by ecosystem
  • Note: Your in-game name will automatically sync with your Ely.by username

Offline (Singleplayer)

  • Enter any name in the Username box on the Home screen
  • Click Launch to play immediately
  • No online skin support

3. Skin System

MeoLauncher integrates deeply with Ely.by to provide the best skin experience for Beta 1.7.3.

How to Change Skin

  1. Go to the Skin tab
  2. Login to your Ely.by account (if not already)
  3. Click Manage Skins on Ely.by to open the Ely.by website in your browser
  4. Choose a skin you like and click "Wear" on the Ely.by website
  5. Return to the launcher and click Launch

Capes

Ely.by capes are also supported and will appear automatically if equipped on the web.

4. Settings & Performance

For the smoothest experience, consider these settings:

Settings Tab

RAM Allocation: Slider to select RAM amount

  • 8GB RAM PC: Select 2GB - 3GB
  • 16GB+ RAM PC: Select 4GB
  • Note: Do not allocate too much RAM (over 6GB) as it may cause lag spikes due to Java Garbage Collector

Performance Tab (Important)

These tweaks help reduce lag for the old Beta version:

  • Reduce Micro-stutters (Recommended): Uses CMS Garbage Collector. Makes the game much smoother on Beta 1.7.3
  • G1 Garbage Collector: More modern algorithm, good for high-end PCs
  • Memory Pre-Touch: Reduces stuttering when first loading worlds
  • Aggressive Inlining: Increases CPU processing speed

5. Troubleshooting

Game won't start?

  • Check Java Path in Settings
  • Try reducing RAM to 1024MB or 2048MB
  • Disable some options in the Performance tab

Skin not showing?

  • Ensure you are logged in to Ely.by in the Skin tab
  • In-game Username must match your Ely.by account name exactly
  • Check the launcher console for "Authlib-Injector enabled" message

"Bad Video Card Drivers" error?

  • Update your graphics card drivers to the latest version
  • Enable Hardware Acceleration in Windows

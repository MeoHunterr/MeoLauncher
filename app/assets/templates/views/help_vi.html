Hướng dẫn sử dụng MeoLauncher

Chào mừng bạn đến với MeoLauncher, launcher tối ưu hóa chuyên biệt cho Minecraft Beta 1.7.3. Tài liệu này sẽ hướng dẫn bạn sử dụng toàn bộ tính năng của launcher.

  3. Hệ thống Skin

  5. Khắc phục sự cố

1. Bắt đầu

MeoLauncher được thiết kế để đơn giản hóa việc chơi Minecraft Beta 1.7.3 với các tính năng hiện đại.

Yêu cầu hệ thống

  • Java: Java 8 hoặc mới hơn (Launcher tự động phát hiện)
  • RAM: Tối thiểu 1GB, khuyến nghị 2GB-4GB
  • Hệ điều hành: Windows 10/11 (64-bit)

Giao diện chính

  • Home: Nơi đăng nhập và khởi động game
  • Screenshots: Quản lý ảnh chụp màn hình trong game
  • Texture Packs: Cài đặt và quản lý gói tài nguyên
  • Skin: Quản lý skin và áo choàng Ely.by
  • Settings: Cài đặt chung cho launcher và game
  • Performance: Các tùy chỉnh tối ưu hóa chuyên sâu

2. Tài khoản & Đăng nhập

MeoLauncher hỗ trợ 3 chế độ đăng nhập:

Microsoft (Bản quyền)

  • Nhấn nút Login with Microsoft ở màn hình Home
  • Làm theo hướng dẫn để xác thực qua trình duyệt
  • Hỗ trợ đầy đủ skin và multiplayer server bản quyền

Ely.by (Hệ thống Skin)

  • Nhấn nút Login with Ely.by ở màn hình Home hoặc vào tab Skin
  • Nhập username/email và mật khẩu Ely.by
  • Hỗ trợ skin, áo choàng và danh sách bạn bè từ hệ sinh thái Ely.by
  • Lưu ý: Tên trong game sẽ tự động đồng bộ theo tên Ely.by của bạn

Offline (Chơi đơn)

  • Nhập tên bất kỳ vào ô Username ở màn hình Home
  • Nhấn Launch để chơi ngay lập tức
  • Không hỗ trợ skin online

3. Hệ thống Skin

MeoLauncher tích hợp sâu với Ely.by để mang lại trải nghiệm skin tốt nhất cho Beta 1.7.3.

Cách đổi Skin

  1. Vào tab Skin
  2. Đăng nhập tài khoản Ely.by (nếu chưa)
  3. Nhấn nút Quản lý Skin trên Ely.by để mở trang web Ely.by trong trình duyệt
  4. Chọn skin bạn thích và nhấn "Wear" (Mặc) trên trang Ely.by
  5. Quay lại launcher và nhấn Launch

Áo choàng (Capes)

Áo choàng Ely.by cũng được hỗ trợ và tự động hiển thị nếu bạn đã trang bị trên web.

4. Cài đặt & Hiệu năng

Để có trải nghiệm mượt mà nhất, hãy tham khảo các thiết lập sau:

Tab Settings

RAM Allocation: Kéo thanh trượt để chọn lượng RAM

  • Máy 8GB RAM: Chọn 2GB - 3GB
  • Máy 16GB+ RAM: Chọn 4GB
  • Lưu ý: Không nên chọn quá nhiều RAM (trên 6GB) vì có thể gây giật lag do Java Garbage Collector

Tab Performance (Quan trọng)

Đây là các tweak giúp giảm lag cho phiên bản Beta cũ:

  • Giảm Giật Nhỏ (Khuyên dùng): Sử dụng thuật toán dọn rác CMS. Giúp game mượt hơn rất nhiều trên Beta 1.7.3
  • G1 Garbage Collector: Thuật toán hiện đại hơn, tốt cho máy cấu hình cao
  • Memory Pre-Touch: Giúp giảm khựng khi mới vào thế giới
  • Aggressive Inlining: Tăng tốc độ xử lý của CPU

5. Khắc phục sự cố

Game không khởi động?

  • Kiểm tra lại Java Path trong Settings
  • Thử giảm lượng RAM xuống 1024MB hoặc 2048MB
  • Tắt bớt các tùy chọn trong tab Performance

Skin không hiện?

  • Đảm bảo bạn đã đăng nhập Ely.by trong tab Skin
  • Tên trong game (Username) phải trùng khớp hoàn toàn với tên tài khoản Ely.by
  • Kiểm tra console launcher có dòng "Authlib-Injector enabled" không

Lỗi "Bad Video Card Drivers"?

  • Cập nhật driver card đồ họa của bạn lên phiên bản mới nhất
  • Bật Hardware Acceleration trong Windows
